# Hotfix: Chart Color + Confirm Cancel on New Document

## Scope
1. ปรับสีกราฟรายงานให้มีสีสันและอ่านง่ายขึ้น โดยเฉพาะกราฟโดนัท "ยอดขายแยกประเภทสินค้า/บริการ"
2. เพิ่มการยืนยันก่อนยกเลิกการสร้างเอกสารใหม่ หากมีการกรอกข้อมูลแล้ว

## Files changed
- frontend/src/js/app.js

## Details
- เพิ่มชุดสีสำหรับกราฟ (`chartPalette()`)
- ปรับสีของ advanced report charts:
  - `advanced-revenue-chart`
  - `advanced-sales-type-chart`
  - `advanced-aging-chart`
- เพิ่ม snapshot ของฟอร์มสร้างเอกสารใหม่
- เมื่อผู้ใช้กดยกเลิก/กดปิด/กด ESC ระหว่างสร้างเอกสารใหม่และมีข้อมูลที่แก้ไขแล้ว ระบบจะถามยืนยันก่อนปิด
- ถ้าบันทึกเอกสารสำเร็จ ระบบจะปิด modal แบบไม่ถามซ้ำ

## QA checklist
- เปิดหน้ารายงานขั้นสูงและตรวจว่ากราฟมีสี
- ตรวจกราฟโดนัทไม่แสดงสีเทาอย่างเดียว
- เปิดสร้างเอกสารใหม่ กรอกข้อมูลบางส่วน แล้วกดยกเลิก -> ต้องมี confirm
- เปิดสร้างเอกสารใหม่แล้วไม่กรอกอะไร กดยกเลิก -> ปิดได้ตามปกติ
- กด ESC ระหว่างสร้างเอกสารใหม่หลังแก้ไขข้อมูล -> ต้องมี confirm
