# Hotfix: Audit Log Readable Details

## Scope
ปรับหน้า Audit Log ให้แสดงรายละเอียดเป็นข้อความภาษาไทยอ่านง่าย แทนการแสดง JSON ดิบเพียงอย่างเดียว

## Files changed
- frontend/src/js/app.js
- frontend/src/css/styles.css

## Details
- เพิ่ม formatter สำหรับ Action เช่น CREATE → สร้าง, UPDATE_STATUS → เปลี่ยนสถานะ
- เพิ่ม formatter สำหรับ Entity เช่น document → เอกสาร, customer → ลูกค้า
- แปลงรายละเอียดเอกสารให้เป็นข้อความ เช่น
  - สร้างใบเสร็จรับเงิน RC001-6907 ยอดรวม ฿3,000.00
  - เปลี่ยนสถานะเอกสาร QT001-6907 จาก รอดำเนินการ เป็น อนุมัติแล้ว
- เพิ่ม chip แสดงชนิดเอกสาร เลขเอกสาร ยอดรวม และสถานะ
- ยังสามารถกด “ดูข้อมูลดิบ” เพื่อเปิด JSON ดั้งเดิมได้สำหรับตรวจสอบเชิงเทคนิค

## QA checklist
1. เข้าเมนู Audit Log
2. ตรวจว่า Action แสดงเป็นภาษาไทย
3. ตรวจว่า Entity แสดงเป็นภาษาไทย
4. ตรวจว่ารายละเอียดไม่แสดงเป็น JSON ยาว ๆ ทันที
5. กด “ดูข้อมูลดิบ” แล้วต้องเห็น JSON เดิม
